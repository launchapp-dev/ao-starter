# ao-starter — Product Vision

Build create-ao CLI that scaffolds AO workflows for any project. Detect project type (TS monorepo, Rust, Python), generate custom.yaml with appropriate agents, phases, schedules. Support: ao init, ao init --template nextjs, ao init --template rust.
